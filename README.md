# Aeterna AI Landing Page

Professional landing page for Aeterna AI services targeting small businesses.

## Features

- Modern, responsive design
- Contact form with Google Sheets integration
- Professional layout optimized for conversions
- Fast loading times

## Technologies Used

- Node.js
- Express.js
- EJS templating
- Google Sheets API
- HTML5, CSS3, JavaScript

## Setup Instructions

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file with your Google Sheets configuration
4. Run the application: `npm start`

## Environment Variables

- `GOOGLE_SHEET_ID`: Your Google Sheet ID

Additionally, place your Google service account key in a file named `google-service-account.json` in the root directory of the project.
